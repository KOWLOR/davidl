iGoogle va fermer. Je trouve ça moche. J'utilise ce service depuis le début, étant fan de la simplicité de ce lecteur de flux RSS. Mais toutes les belles choses ont une fin, j'ai du trouver une solution pour pouvoir continuer à suivre l'actualité de mes sites préférés.

Sauvegarder vos flux

Convertir le format iGoogle

Le format de base pour l'importation / exportation de flux RSS des agrégateurs est le OPML.

iGoogle propose un export en XML. (C'est peut être pour ça que le service ferme #troll) Voici un site qui vous permettra de faire l'export : Export iGoogle Feeds to OPML.

Si vous rencontrez des problèmes lors de l'export vous pouvez construire entièrement votre fichier OPML. Il vous suffit d'avoir les URL de vos  flux RSS.
