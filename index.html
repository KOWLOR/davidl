Voici ce que je fais

  • Webdesign

    Je crée des applications web et des sites internet. Je développe des sites adaptatifs accessibles quelquesoit le périphérique.
    Découvrez ma façon de travailler →

  • Ecriture

    La meilleure façon de maîtriser un sujet est de l'enseigner aux autres. J'écris des articles sur la conception web mobile, le responsive design et les outils que j'utilise quotidiennement.

  • Partage

    J'aime partager des informations. J'apprends quelque chose de nouveau chaque jour, et je participe à la conversation collective qu'est Twitter.

  • Idée, Papier, Crayon

    Je fais parti de ces designers incapables de commencer à faire quoi que ce soit sur un logiciel avant d’avoir posé mes idées sur papier. Avant je jetais ces maquettes directement à la poubelle, mais ça c’était avant. Maintenant je prends des photos pour conserver mes idées et les partager sur ce site.

  • Guide CSS

    Pendant des années j’ai travaillé sur des projets où le CSS peut être comparé à un bidonville : un espace où s’entasse dans un chaos le plus complet divers éléments. J’ai rédigé ce guide qui regroupe les bonnes pratiques pour coder des feuilles de styles de qualité.

  • Pokemon breakpoint

    Les développeurs front-end ont un problème dans la conception adaptative. Nous fixons des points d’arrêt en CSS et peut-être que nous voulons les utiliser dans notre Javascript. Pokemon breakpoint est une façon amusante de faire le lien entre ces 2 mondes.

  • Responsive icon

    Remplacer des icônes en images par des symboles HTML présente de nombreux avantages. J’ai développé cette application pour rapidement copier / coller ces caractères utiles en webdesign.

  • Hoooooo

    Coder des sites web à longueur de temps peut vous griller les neurones. J’ai créé ce site pour détendre l’atmosphère lorsqu’une fonctionnalité / un site web plante.

  • Webcitation

    Jours après jours, je collecte des citations à travers de nombreux livres, articles de blog… J’administre webcitation pour toutes les regrouper et les traduire pour vous.
